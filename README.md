# SSR React/TSX example

## Package tech used
- React
- Typescript
- Express

## Installing the app dependencies
```
$ npm install
```

## Building the app
```
$ npm run build
```

## Running the application
```
$ npm start
```
You should see "Server listening at http://localhost:3636" shortly, which means the server is running.


## Routes to hit
- /
- /home
- /about
- /any-other-non-existent-route
